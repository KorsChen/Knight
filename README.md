## Usage

```js
cd ~/.cocoapods/repos/master/Specs/1/9/2/libwebp
sudo vim libwebp.podspec.json
// 将其中的"git" 对应的url替换为https://github.com/webmproject/libwebp.git,并保存
pod install
```
